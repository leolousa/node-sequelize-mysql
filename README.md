# Node Sequelize e MySQL

> Aplicação exemplo de uso de API NodeJS e Express com Sequelize e MySQL

## Instalação

1. Iniciar o NodeJS
```sh
npm init
```
2. Instalar o Express
```sh
npm i --save express
```
3. Instalar o Sequelize
```sh
npm i --save sequelize
```
4. Instalar o MySQL2
```sh
npm i --save mysql2
```
5. Instalar o Nodemon
```sh
npm i nodemon -D --save
```
